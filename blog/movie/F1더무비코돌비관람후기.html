# 영화 맛있게 보려면

  1. 인생에 사고를 좀 겪어야함
  2. 실패 경험이 있어야함
  3. 나이가 좀 있어야함 최소 40 이상
  4. 테토남 테토녀

f1 직관 경험 + f1 탑승 경험 + f1 팀 멤버의 열정과 환희 를 느낄 수 있음..

f1 실제로 가보지도 않았고 별로 관심도 없지마능..

하지만 와타시는 거친세상을 홀로 헤쳐나가는 테토녀라 1번 2번 4번 조건 충족하므로

마지막 질주 장면에서 진짜 눙물 고여가며 심장 쪼그라 들면서 봤음열 (fly...!)
