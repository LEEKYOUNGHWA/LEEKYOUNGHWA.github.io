# 이클립스 자바 컴파일

요즘도 CI/CD 가 없는 시스템이... 있... 네?

이클립스는 기본적으로 java를 컴파일 할 수 있는 환경입니다.

소스를 저장할 때 자동으로 컴파일되도록 기본 설정되어 있습니다.

자동 컴파일은 메뉴의 Project > Build Automatically 에 v표시가 되어 있을 때만 작동합니다.

수동 컴파일은 메뉴 Project의 Build Project나 Build A을 사용하면 됩니다. 자동 컴파일로 설정되어 있을 경우 위 메뉴는 비활성화됩니다.

실행은 해당 소스 열어놓고 메뉴의 Run > Run As > Java Application 클릭하면 됩니다.
