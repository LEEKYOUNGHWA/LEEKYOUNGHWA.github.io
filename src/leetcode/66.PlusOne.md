# 66. Plus One

```java
class Solution {
    public int[] plusOne(int[] digits) {

        for (int i = digits.length - 1; i >= 0; i--) {
            if (digits[i] < 9) {
                digits[i]++;
                return digits;
            }
            digits[i] = 0;
        }

        // 여기까지 왔다는 건 전부 9였다는 뜻
        int[] result = new int[digits.length + 1];
        result[0] = 1;
        return result;
    }
}
```
