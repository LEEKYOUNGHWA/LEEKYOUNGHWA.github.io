# 타입 단언 (Type Assertion)

실제로 타입을 바꾸는것은 아니다. 컴파일러에게 잠시 단언해서 오류를 피함

타입 단언의 규칙

  • 값 as 단언 <- 단언식
  • A as B
  • A가 B의 슈퍼타입이거나
  • A가 B의 서브타입이어야 함
type Person = {
  name: string;
  age: number;
};

let person = {} as Person;
person.name = "이정환";
person.age = 27;
